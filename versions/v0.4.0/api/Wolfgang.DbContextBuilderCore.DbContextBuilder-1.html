    Class DbContextBuilder<T>

    Uses the Builder pattern to create instances of DbContext types seeded with specified data.

    Remarks

    When using the SQLite provider, the builder holds an open SQLite in-memory connection. Dispose the builder only after all DbContext instances returned by BuildAsync() are no longer in use, as disposing the builder closes the shared connection and destroys the in-memory database.
