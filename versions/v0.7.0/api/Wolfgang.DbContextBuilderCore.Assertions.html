    Namespace Wolfgang.DbContextBuilderCore.Assertions

    Classes

    DbContextAssertionException

    Thrown when a DbSetAssertions<TEntity> assertion fails. Carries an EF-aware failure message that identifies the entity type and the specific check that did not pass.

    DbContextAssertionsExtensions

    Entry-point extensions that produce a DbSetAssertions<TEntity> from a DbSet<TEntity> or IQueryable<T>. Intended to be imported via using Wolfgang.DbContextBuilderCore.Assertions; in test files so that context.Set<Product>().Should() resolves.

    DbSetAssertions<TEntity>

    Fluent, chainable assertions over an IQueryable<T> — either a DbSet<TEntity> directly or a filtered query derived from one — for use in DbContextBuilder-backed tests. Each method runs the corresponding query asynchronously, throws DbContextAssertionException with an EF-aware message on failure, and returns this so the next assertion can be chained.
